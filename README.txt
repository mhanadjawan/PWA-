Flashcards PWA - Offline package
Files included:
- index.html
- settings.html
- styles.css
- app.js
- manifest.json
- service-worker.js
- assets/icons/icon-192.png, icon-512.png

How to run locally (XAMPP):
1. Copy the folder 'flashcards_full_pwa' into C:\xampp\htdocs\
2. Start Apache from XAMPP Control Panel
3. Open http://localhost/flashcards_full_pwa/index.html
4. On first load you'll see a login modal. Create a new user or login using created user.
5. Use Settings to manage users, import/export, change theme, test notifications.

Notes:
- All data stored locally (localStorage) and works offline.
- Passwords are hashed with SHA-256 in the browser.
- PWA: you can install the app if browser supports it.
